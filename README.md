# TO DO:
Assurez vous d'avoir téléchargé tous les fichiers dans l'arborescance suivante :
* `game.py`
* img 
    * ``spriteLeft-1.png``
    * ``spriteLeft-2.png``
    * ``spriteRight-1.png``
    * ``spriteRight-2.png``
    * ``spritePlayer-1.png``
    * ``spritePlayer-2.png``
* music
    * ``home.mp3``
    * ``ride.mp3``

Il ne vous restera plus qu'à éxécuter ``game.py``

# HOW TO PLAY :
Une fois le TO DO complété, il ne reste plus qu'à run `main.py`. 

* Si un erreur a lieu, vérifiez que l'arborescance et bien respectée. 

* Si vous n'arrivez pas à controlez votre voiture, vérifiez que vous appuyez bien sur les flèches ou les touches `z` et `s`. 

Pour tout autre problème, contactez moi personellement.
